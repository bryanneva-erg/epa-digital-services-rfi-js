## Installation

```bash
git clone https://github.com/Eastern-Research-Group/epa-digital-services-rfi-js.git
cd epa-digital-services-rfi-js
npm install
npm start
open http://localhost:3000
```

Transforms are enabled for files inside `src` (except `index.js`).
